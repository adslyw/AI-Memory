# M3U Player - Chrome 扩展版本

## 安装步骤

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的 **开发者模式**
3. 点击左上角的 **加载已解压的扩展程序**
4. 选择此目录：`/home/deepnight/.openclaw/workspace/projects/m3u-player-extension`
5. 扩展安装完成，工具栏会出现播放器图标

## 使用说明

### 基本操作
- 点击工具栏图标打开播放器
- 点击 "+ 添加频道" 输入频道名称和 M3U8 链接
- 在列表中点击频道名称进行播放
- 拖拽频道可以重新排序
- 使用搜索框快速过滤频道
- 点击侧边栏左侧的箭头按钮可以收起/展开列表
- 点击右上角月亮图标切换深色主题

### 跨域播放
此插件已声明 `host_permissions`，可以播放跨域 M3U8 链接。如果某些链接仍无法播放，请检查：
- 链接是否有效且可访问
- 网络连接正常
- HLS.js 不支持某些加密流

### 数据存储
所有频道数据保存在 Chrome 本地存储中（`chrome.storage.local`），不会上传到任何服务器。

### 导入旧数据
如果你有旧版 Web 版本的 LocalStorage 数据（JSON 格式）：
1. 点击添加按钮上方的 "检测到旧数据" 提示（如果首次使用无数据，此提示可能不会显示）
2. 将 JSON 数据粘贴到输入框并点击导入

### 注意事项
- 本扩展为开发测试版本，不发布到 Chrome Web Store
- 可以在 `chrome://extensions/` 中随时卸载或禁用
- 如需备份数据，可使用 Chrome 存储查看工具导出 `m3u_channels` 键值

## 技术架构

- Manifest V3
- 使用 HLS.js 进行流媒体播放
- chrome.storage.local 持久化
- 完全在本地运行，无需后端服务

## 故障排除

**问题：点击播放无反应**
- 检查控制台是否有错误
- 确认 M3U8 链接可访问（在浏览器中直接打开）

**问题：视频卡顿**
- 检查网络带宽
- 尝试其他频道测试

**问题：无法添加频道**
- 检查是否输入了名称和链接
- Chrome 存储是否已满（极罕见）
